# mp1-2026-1
Центральный репозиторий практики группы 3825Б1ПМ2-1 по курсу МП1 (весенний семестр)

## Общее описание

- Практические задания для выполнения в т.к. и дома разделены на 6 блоков.
- Блоки, по возможности, выстроены по нарастанию сложности.
- Во всех заданиях предполагается разработка класса (классов) и консольной программы с примером (примерами) использования.
- В блоках 1-5 по 6 вариантов заданий. Распределение вариантов между студентами группы выполняется преподавателем.
- Задания в блоках 1-2 требуют разработки одного класса. На выполнение заданий этих блоков дается по одной неделе.
- Задания в блоках 3-4 допускают (но не требуют) разработку системы классов. На выполнение заданий этих блоков дается по две недели.
- Задания в блоке 5 требуют разработки не менее двух взаимодействующих друг с другом классов. На выполнение задания этого блока дается 3 недели.
- Задания в блоке 6 требуют разработки системы классов и законченного консольного приложения. На выполнение задания этого блока дается 3 недели.

Прием заданий выполняется с использованием СКВ git и сайта github. Задание считается сданным, если pull request в upstream-репозиторий преподавателя одобрен.

Формулировки заданий будут выкладываться в отдельных ветках (task1, rask2 и т.д.).

Вы работаете исключительно в ветке main. Другие ветки не создавать!

Начальные действия по работе с репозиторием см. в Задании 0 осеннего семестра.
